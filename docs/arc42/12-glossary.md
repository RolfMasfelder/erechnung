# 12. Glossary

| Term                  | Definition                                                                                   |
|-----------------------|----------------------------------------------------------------------------------------------|
| **ADR**               | Architecture Decision Record - Document capturing an important architectural decision        |
| **API**               | Application Programming Interface - Interface that allows software components to communicate |
| **Django**            | Python web framework used for the backend application                                       |
| **DRF**               | Django REST Framework - Extension for building RESTful APIs in Django                       |
| **eRechnung**         | Electronic Invoice - Digital representation of an invoice                                   |
| **EN16931**           | European Standard for electronic invoicing                                                  |
| **GoBD**              | Principles for properly maintaining and storing books, records, and documents in electronic form and for data access |
| **GDPR**              | General Data Protection Regulation - EU regulation on data protection and privacy           |
| **JWT**               | JSON Web Token - Compact, URL-safe means of representing claims between two parties         |
| **ORM**               | Object-Relational Mapping - Technique for converting data between incompatible systems      |
| **PDF/A-3**           | PDF format for long-term archiving that allows embedding of arbitrary file formats          |
| **PostgreSQL**        | Open-source relational database management system                                           |
| **REST**              | Representational State Transfer - Architectural style for distributed systems                |
| **ZUGFeRD**           | Zentraler User Guide des Forums elektronische Rechnung Deutschland - German e-invoicing standard |
